# bulkEmailProject
Easily Manage and Send group Emails with a single Click, reaching all your Contacts instantly through a clean and user-friendly interface.

---

## 🚀 Features
- **Home Page** (`index.php`) – Page d’accueil avec navigation vers les fonctionnalités principales.  
- **User Registration** (`signup.php)`) – Création d’un compte utilisateur sécurisé.  
- **User Login** (`login.php`) – Authentification des utilisateurs enregistrés.  
- **Main Dashboard** (`main.php`) – Envoi d’un message à plusieurs destinataires en un seul clic. 
- **Recipients Management** (`ajouterSupprimer.php`) – Ajout et suppression des emails destinataires.  
- **Sent Messages History** (`historique.php`) – Consultation des messages envoyés.  
- **Edit / Delete Messages** (`modifier_message.php`) – Modification et suppression des messages..  
- **Logout** (`logout.php`) Déconnexion sécurisée de l’utilisateur.  
- **Database Integration** – MySQL database (`ldstrtz.sql`) for persistence.  
- **Custom Design** – Responsive styles via `css/style.css` and custom fonts.
