# bulkEmailProject
Easily Manage and Send group Emails with a single Click, reaching all your Contacts instantly through a clean and user-friendly interface.

---

## 🚀 Features
- **Home Page** (`index.php`) – Page d’accueil avec navigation vers les fonctionnalités principales.  
- **User Registration** (`signup.php)`) – Création d’un compte utilisateur sécurisé.  
- **Shop** (`menu_pages/Shop`) – Browse and purchase items related to the festival.  
- **Orders Management** (`menu_pages/orders`) – View and manage placed orders.  
- **Media Section** (`menu_pages/Media`) – Access media content and resources.  
- **Accessibility Page** (`menu_pages/Acessibilty`) – Accessibility information for attendees.  
- **Privacy Policy** (`menu_pages/Privacy_policy`) – Legal and privacy details.  
- **Authentication** – Login/Logout (`logout.php`) system for users.  
- **Database Integration** – MySQL database (`ldstrtz.sql`) for persistence.  
- **Custom Design** – Responsive styles via `css/style.css` and custom fonts.
